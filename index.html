Legal PDF e-signatures in one line of code

eIDAS Compliant PKCS#7 X.509 RFC 3161 MIT License

⚡ Overview

A production-ready SDK for signing and verifying PDFs with cryptographic compliance (AdES-B-LT). Built for enterprise-grade digital signature workflows.

📦 Install

npm install @pdf-signature/core
pnpm add @pdf-signature/core
yarn add @pdf-signature/core

✍️ Sign a PDF

import { pdfSign } from '@pdf-signature/core';

const result = await pdfSign({
  pdfBuffer: fs.readFileSync('./contract.pdf'),
  signerInfo: {
    name: "Alice Johnson",
    email: "alice@example.com",
    organization: "Acme Corp"
  },
  apiKey: process.env.PDF_SIGNATURE_API_KEY!
});

fs.writeFileSync('./signed.pdf', result.signedPdfBuffer);

🔍 Verify Signature

import { pdfVerify } from '@pdf-signature/core';

const res = await pdfVerify({
  pdfBuffer: fs.readFileSync('./signed.pdf'),
  apiKey: process.env.PDF_SIGNATURE_API_KEY!
});

console.log(res.isValid);
